Общее описание
Разработать автономное приложение на Python для автоматизированной генерации ценников форматом 80×45 мм и 45×80 мм из данных Excel/CSV, с оптимальной раскладкой на листах A4/A3 с учетом ориентации и плотности, экспортом итоговых файлов в PDF и CDR.

Основные требования
Входные данные
Excel или CSV файл с колонками:

name — название блюда

price1, price2, price3 — цены (необязательно все)

weight1, weight2, weight3 — веса (необязательно все)

kcal, protein, fat, carbs — КБЖУ (опционально)

Обработка данных
Читать данные, обрабатывать отсутствие значений.

Генерировать ценники по шаблону:

Оранжевый фон

Логотип в верхней части

Название, цена, вес, КБЖУ с переносами и адаптацией шрифтов

Поддержка 3 вариантов: одинарный, двойной, тройной ценники по цене/весу.

Раскладка и оптимизация
Рассчитывать 4 варианта раскладки:

A4 горизонталь

A4 вертикаль

A3 горизонталь

A3 вертикаль

Выбирать оптимальный по максимальному количеству размещенных ценников.

Максимально плотно размещать ценники без масштабирования.

Вывод
Генерировать один или несколько листов, экспортировать в PDF (300 DPI, реальный размер).

Создавать файл CDR с отдельными объектами для каждого ценника.

Опционально — экспортировать PNG/JPG изображений ценников.

Общие требования
Исполняемый файл (EXE), собранный с помощью PyInstaller или аналога.

Работать без интернета, без подписок.

Простота интерфейса (поддержка командной строки или GUI по желанию).

Обработка ошибок и уведомления о некорректных данных.

Этапы работы
Анализ данных и подготовка шаблонов.

Реализация генерации ценников.

Реализация алгоритмов раскладки.

Экспорт в нужные форматы.

Тестирование и наладка.
